kfactory.utils.geo.extrude_path(target, layer, path, width, enclosure=None, start_angle=None, end_angle=None)[source]#

Extrude a path from a list of points and a static width

Parameters

  • target (KCell) – the cell where to insert the shapes to (and get the database unit from)

  • layer (LayerEnum | int) – the main layer that should be extruded

  • path (list[DPoint]) – list of floating-points points

  • width (float) – width in µm

  • enclosure (Optional[Enclosure]) – optoinal enclosure object, specifying necessary layers.this will extrude around the layer

  • start_angle (Optional[float]) – optionally specify a custom starting angle if None will be autocalculated from the first two elements

  • end_angle (Optional[float]) – optionally specify a custom ending angle if None will be autocalculated from the last two elements

Return type

None
